# Glam Trash and Accessories

Glam trash and accessories are recurring motifs that add:
- tacky luxury
- ironic diva energy
- color accents
- visual clutter
- lowbrow opulence
- absurd status symbols

This file defines how these elements function in `laser_cat_style`.

---

# Core principle

Glam trash is not elegant.
It is:
- rhinestone
- glitter
- oversized
- neon
- plastic
- fake-gold
- costume-jewelry
- dollar-store diva
- internet tacky

The cat is not actually glamorous.
The cat is wearing absurd glam as a joke.

---

# Why glam trash works

Glam trash is powerful because it:
- adds color
- adds irony
- signals lowbrow taste
- contrasts with cosmic or food contexts
- creates diva-goblin energy
- is instantly readable
- works as floating accent or worn accessory
- feels internet-native

A cat with sunglasses and diamonds is immediately funny.
A cat with a bikini in space is absurdly iconic.

# Placement modes

## Worn
The accessory is on the cat.

Examples:
- sunglasses on face
- hat on head
- bikini on body
- chain around neck

## Floating nearby
The accessory hovers as separate object.

Examples:
- diamond near cat
- sunglasses floating
- hat nearby
- glam debris cloud

## Scattered
Multiple accessories distributed across the scene.

Examples:
- diamond scatter
- glam object cloud
- accessory constellation

## Accessory larger than cat

# Common failure modes

## Too elegant
The glam becomes actually tasteful.

Fix:
- increase tackiness
- add neon colors
- add rhinestones
- reduce refinement
- add absurd context

## Too dominant
The accessories overpower the cat.

Fix:
- reduce accessory size
- simplify the glam set
- increase cat presence
- make accessories floating rather than worn

## Generic fashion drift
The image becomes fashion illustration with no cat-stupid humor.

Fix:
- add cat idiocy
- add absurd context
- add cosmic background
- reduce fashion seriousness

## Under-integrated
The accessories look pasted on without relation to the scene.

Fix:
- add glow or interaction
- position for compositional flow
- make accessories part of object cloud
- add narrative suggestion
