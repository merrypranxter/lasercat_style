# Psychedelic Eyes and Drug Cat

The psychedelic/drug-cat branch of `laser_cat_style` overlaps with stoner meme culture, altered-state visual aesthetics, and psychedelic internet imagery.

This file defines how that branch functions while remaining distinct from generic trippy cat art.

---

# Core principle

The drug-cat branch is not primarily about drug advocacy.
It is about:
- visual overload
- altered-state energy
- cosmic stupidity
- chemical-cat iconography
- meme escalation
- color violence
- fake transcendence

The cat is not a spiritual guide.
The cat is a chemical mess.

---

# Why this branch matters

Drug-cat imagery is a major part of the style because it provides:
- easy visual escalation
- recognizable iconography
- rainbow/neon color behavior
- altered-state meme shorthand
- posterized face closeups
- internet-native humor

But without structure, it easily drifts into:
- generic trippy cat art
- weed-joke spam
- psychedelic poster clichés
- unfunny color chaos

This file exists to keep the branch grounded in the style's core grammar.

---

# Visual markers of the branch

## 1. Glowing or altered eyes

The eyes become the focal point.

Common forms:
- oversized irises
- bright saturated color
- geometric or prismatic patterns
- starburst or radiating effects
- posterized color bands
- black-light glow
- neon-edge emphasis

The eyes should feel:
- chemically intensified
- visually loud
- slightly wrong
- comedic rather than truly mystical

---

## 2. Face posterization and color burn

The cat face is often treated with:
- oversaturated color
- posterized or quantized color bands
- high-contrast color shifts
- unnatural hue mapping
- color clipping
- glow bleed

The result is:
- less naturalistic
- more meme-like
- more internet-crude
- more visually aggressive

---

## 3. Acid-tab and drug iconography

Common icons include:
- acid tabs or blotter paper
- mushrooms
- weed leaves
- chemical symbols
- pill imagery
- drug-adjacent props

These should be used as:
- visual punchlines
- color accents
- meme signifiers

not as:
- serious drug advocacy
- tasteful psychedelic decoration

---

## 4. Aura and glow effects

The cat may be surrounded by:
- color halos
- light bleed
- rainbow edges
- chromatic aberration
- glow clouds
- neon outlines

These effects increase:
- visual intensity
- fake transcendence
- internet-trash energy

---

## 5. Cosmic or void backgrounds

Drug-cat imagery often pairs with:
- galaxy fields
- black void stage
- neon gradients
- star clouds
- color-bleed environments

The background supports the "cosmic" part of chemically-cosmic stupidity.

# Cat types best suited for this branch

Best candidates:
- big head crops
- face-forward cats
- wide-eyed cats
- slightly alarmed cats
- blank cats that can handle color abuse
- cats with readable facial structure

Also viable:
- cats with visible pupils that can be intensified
- cats with expressive eyes that can be made ridiculous

Weaker candidates:
- cats with tiny eyes
- heavily side-facing cats unless the scene is very strong
- cats already too kawaii or too serious without additional corruption

Drug-cat imagery tends to produce:

the cat looks too innocent or too dumb for the intensity of the treatment.

## Good cheapness
- posterized color bands
- crude glow
- oversaturated eye treatment
- slightly blown-out highlights
- obvious digital manipulation
- internet-trash artifact energy

## Bad cheapness
- muddy colors
- unreadable eyes
- effects that obscure the cat face entirely

# Common failure modes

## Generic trippy drift
The image becomes sincere psychedelic art with no cat-stupid humor.

Fix:
- add cat idiocy
- add meme-native tackiness
- reduce mystical intensity
- reintroduce internet-trash energy

## Weed-joke spam
The image is just "cat + weed leaf" with no broader style logic.

Fix:
- add collage assembly
- add color escalation
- add cosmic background
- add object scatter
- add eye treatment

## Overly sincere transcendence
The cat looks like a genuine spiritual guide.

Fix:
- make the cat dumber
- add absurd context
- add tacky glam objects
- add food or household nonsense
- reduce the mystical framing

## Effect dominance
The treatment overpowers the cat's identity.

Fix:
- simplify the eye treatment
- preserve facial readability
- reduce glow intensity
- make the cat face larger in frame
