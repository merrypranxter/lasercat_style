# Polished vs Correctly Bad

This file defines the crucial distinction between polish and correct badness in `laser_cat_style`.

## The polish trap

Modern image systems default toward:
- Cleanliness
- Refinement
- Seamlessness
- Realism
- Tasteful restraint

These are fatal to `laser_cat_style`.

## Why polish kills the style

### Loses collage visibility
Seamless integration removes the assembled feel.

### Loses internet energy
Polish signals professional production, not meme culture.

### Loses tackiness
Refinement contradicts camp excess.

### Loses authenticity
Clean images feel manufactured, not found.

### Loses humor
The joke depends on some wrongness.

## Correct badness

The style wants:
- Visible assembly
- Tacky excess
- Internet texture
- Meme-native energy
- Deliberate wrongness

## The balance

```
Too polished (drift) ← CORRECT ZONE → Too ugly (unreadable)
```

The correct zone is:
- Rough but readable
- Tacky but iconic
- Crusty but clear
- Wrong but functional

## Examples of correct badness

- Cutout with slight halo
- Glow that bleeds too much
- Colors that clip
- Compression artifacts
- Obvious source mismatch
- Absurd scale
- Tacky accessories

## Examples of wrong badness

- Unreadable silhouette
- Cat completely obscured
- Muddy colors
- Random clutter
- Genuinely unpleasant
- No focal point

## Best practices

### Preserve some roughness
Don't clean too much.

### Embrace tackiness
More is more.

### Maintain readability
Wrong but functional.

### Avoid perfection
Perfection is the enemy.

### Check for drift
Polish is a warning sign.

## Repair strategies

If too polished:
- Add visible cutout edges
- Add glow or halo
- Add compression artifacts
- Reduce seamless integration
- Increase tackiness
