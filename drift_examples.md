# Drift Examples

Examples of drift and how to fix them.

## Example 1: Polished Drift

### Drifted Image
A beautifully rendered cat on a pizza in a photorealistic galaxy, perfect lighting, seamless integration.

### Problem
Too polished, loses collage energy and internet-trash aesthetic.

### Diagnosis
- No visible cutout edges
- Perfect lighting match
- No glow or artifact
- Looks like digital painting

### Fix
- Add visible cutout edges
- Add tacky glow
- Introduce source mismatch
- Reduce seamless integration

### Fixed Description
Cat on pizza with rough cutout edges, tacky glow, obvious compositing, galaxy background with slight color mismatch.

---

## Example 2: Generic Drift

### Drifted Image
A cat sitting in a galaxy with no other elements, no absurdity, just "cat in space."

### Problem
No specific style logic, could be any cosmic cat image.

### Diagnosis
- No recognizable motifs
- No absurd context
- No object nonsense
- No composition mode

### Fix
- Add food items
- Add absurd scale
- Add cosmic debris
- Activate humor engines

### Fixed Description
Cat on giant pizza slice with floating fries, absurd scale, galaxy background, sticker scatter composition.

---

## Example 3: Serious Drift

### Drifted Image
A majestic cat with laser eyes over a city, dramatic cinematic lighting, serious tone.

### Problem
Too epic, loses the stupid humor.

### Diagnosis
- No cat idiocy
- Cinematic treatment
- Serious tone
- No absurd context

### Fix
- Make cat expression blank/confused
- Add tacky elements
- Reduce cinematic drama
- Add food or household objects

### Fixed Description
Blank-faced cat with laser eyes over city, confused expression, fries floating nearby, tacky glow.

---

## Example 4: Cute Drift

### Drifted Image
A very cute kawaii cat with soft colors, gentle glow, no absurdity.

### Problem
Too cute, loses the campy tackiness.

### Diagnosis
- Kawaii aesthetics
- Soft colors
- No stupid premise
- No escalation

### Fix
- Add absurd premise
- Add cosmic escalation
- Add object nonsense
- Increase color violence

### Fixed Description
Cute cat on giant burger with rainbow aura, floating diamonds, neon galaxy, tacky excess.

---

## Example 5: Cool Drift

### Drifted Image
A sleek vaporwave-style cat with clean gradients, minimal elements, design-forward.

### Problem
Too cool, loses the goblin energy.

### Diagnosis
- Vaporwave aesthetics
- Graphic restraint
- No tackiness
- Design-first

### Fix
- Add goblin energy
- Add food/household nonsense
- Increase tackiness
- Add more chaos

### Fixed Description
Cat with shutter shades on pizza, fries and diamonds scattered, chaotic gradient, maximalist.

---

## Example 6: Random Drift

### Drifted Image
A cat with random objects scattered with no hierarchy, no clear joke, visual mush.

### Problem
No structure, no clear premise.

### Diagnosis
- No composition mode
- No cat centrality
- No recognizable motif
- No hierarchy

### Fix
- Establish clear premise
- Strengthen cat centrality
- Add recognizable motif
- Improve hierarchy

### Fixed Description
Cat clearly on pizza throne, fries as supporting elements, galaxy background, single-gag hero.
