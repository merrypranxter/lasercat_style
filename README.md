# laser_cat_style

A visual style system and reference repo for **early-internet cat collage chaos**: poorly or semi-poorly photoshopped cats, cosmic backdrops, rainbow beams, laser eyes, food vehicles, surreal object intrusions, tacky glam trash, and deliberately busted meme-era image logic.

This repo is not just a folder of funny cat images.

It is a structured style system for:
- style definition
- AI prompting
- shader behavior
- generative art systems
- app integration
- visual consistency
- drift prevention

The core premise of `laser_cat_style` is simple:

> **Take a cat. Cut it out badly or semi-badly. Put it somewhere stupid. Add one or more absurd intensifiers. Preserve the wrongness.**

That "wrongness" is not accidental. It is part of the aesthetic DNA.

---

# What this style is

`laser_cat_style` is a family of **internet-native collage aesthetics** centered on cats as chaotic visual agents.

Typical images include:
- cats with laser eyes
- cats puking rainbows
- cats floating through galaxies
- cats riding pizza, sharks, or other absurd mounts
- cats merged with junk food, bread, toilet paper, or household objects
- giant cats towering over cityscapes
- cats surrounded by UFOs, diamonds, burgers, fries, milkshakes, or other random nonsense
- psychedelic cat faces with glowing eyes, oversaturated fur, and drug-adjacent iconography

The style usually feels:
- colorful
- dumb in the correct way
- cosmic
- campy
- tacky
- low-stakes surreal
- collage-native
- chaotic but repeatable

- **structured data** — machine-readable style specs

# Core style formula

A typical `laser_cat_style` image often contains:
1. **a cat**
2. **a collage cutout**
3. **a ridiculous context**
4. **a bright synthetic background or high-contrast scene**
5. **an absurd intensifier**
6. **a preserved sense of cheapness, crust, or incorrectness**

Example intensifiers:
- laser beams
- rainbow emissions
- gigantic scale mismatch
- food vehicle or food throne
- UFOs / planets / cosmic debris
- glam accessories
- drug meme cues
- random floating sticker objects

# Core substyle branches

The repo recognizes several major branches:

## 1. Cosmic Food Cat
Cats on pizza, sandwiches, burgers, floating near fries and milkshakes, wrapped in bread, or seated on absurd edible thrones.

## 2. Laser / Psychedelic / Drug Cat
Laser eyes, rainbow puke, glowing eyes, posterized color burn, acid-tab cat, trippy aura cat.

## 3. Space Collage Cat
Cats in galaxy scenes, nebulae, UFO tableaux, cosmic scatter compositions, glam space nonsense.

## 4. Monster / God Cat
Giant city-dominating cats, ominous destroyer cats, kaiju-cat authority figures, blank-eyed beam gods.

These branches overlap constantly.

# Why this style matters

This style looks dumb, but it is not structurally random.

It has:
- recurring composition grammars
- stable motif families
- repeatable tone logic
- recognizable artifact preferences
- a distinct acceptable quality range
- clear drift and failure modes

Without structure, attempts to reproduce it usually collapse into one of these failures:
- generic space cat art
- generic trippy cat art
- generic meme collage
- overly polished digital poster work
- bland sticker spam
- "funny cat" images with no aesthetic engine

This repo exists to stop that collapse.

---

# What this style is not

It is not:
- polished sci-fi cat illustration
- clean kawaii cat art
- tasteful cosmic pet portraiture
- generic psychedelic visionary painting
- flat mainstream sticker design
- only Nyan Cat derivatives
- sleek vaporwave cat poster art
- random internet nonsense with no repeatable grammar

# How to use this repo

## Human workflow
1. Read the overview files.
2. Learn the substyle branches.
3. Review composition modes, cat power roles, and object logic.
4. Study drift control before prompting.
5. Use prompt templates or structured data to generate outputs.

## AI / app workflow
1. Load `style_spec.json`
2. Load motif, palette, artifact, and scene libraries
3. Select branch + background family + cat role + effect set
4. Apply acceptable ugliness constraints
5. Run drift diagnostics before final output

# Repo philosophy

This repo treats `laser_cat_style` as a **usable visual system**, not just a moodboard.

The goal is not to make images that are merely "cats in weird places."

The goal is to preserve a specific type of:
- internet-era collage stupidity
- cosmic tackiness
- color violence
- object nonsense
- meme-native ugliness
- cat-centered absurd authority

In short:

> **structured stupid**
>
> not random stupid
> not polished stupid
> not sanitized stupid
> **correct stupid**
