# Household Object Nonsense

Household objects are a major motif family in `laser_cat_style`.
They add:
- domestic absurdity
- everyday surrealism
- meme-native recognition
- visual punchlines
- object-merge possibilities
- scatter-mode richness

This file defines how household objects function in the style.

---

# Core principle

Ordinary household items become:
- absurd context
- cat habitat
- visual jokes
- cosmic debris
- meme shorthand
- surreal intrusions

The cat does not just coexist with these objects.
The cat occupies, merges with, or is framed by them.

---

# Why household objects work

Household objects are powerful because they:
- are instantly recognizable
- create domestic-to-cosmic contrast
- feel internet-native
- work as visual punchlines
- are easy to scale absurdly
- support object-merge and hero-gag modes
- add low-stakes surrealism

A cat in bread is iconic.
A cat wrapped in toilet paper is absurd.
A cat with household debris floating in space is meme-native.

# Object-cat relationship types

## Merge
The cat is fused with or framed by the object.

Examples:
- cat in bread
- cat wrapped in toilet paper
- cat in box

## Occupation
The cat sits on or uses the object as platform.

Examples:
- cat on furniture
- cat on box
- cat with device

## Coexistence
The cat and object share space as equals.

Examples:
- cat with household debris cloud
- object scatter compositions

---

# Scale behaviors

## Object larger than cat
Creates absurd habitat or framing.

## Object equal to cat
Creates peer relationship or merge potential.

## Object smaller than cat
Creates scatter richness or debris density.

## Multiple sizes
Creates visual hierarchy and pattern.

# Common failure modes

## Too realistic
The object and cat look like they belong together.

Fix:
- increase scale absurdity
- add cosmic background
- add collage mismatch
- add floating logic
- add more nonsense objects

## Object dominance
The object is more memorable than the cat.

Fix:
- make the cat larger
- make the cat more expressive
- make the cat-object relation more central
- reduce object detail

## Generic domestic scene
The image is just "cat with object" with no absurdity.

Fix:
- change the scale
- change the background
- add cosmic context
- add more objects
- make the object absurdly large or placed

## Under-escalated
The scene is too simple.

Fix:
- add more objects
- add cosmic debris
- add glam accessories
- add glow effects
- add food items
