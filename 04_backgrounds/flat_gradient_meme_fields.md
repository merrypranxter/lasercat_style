# Flat Gradient Meme Fields

Flat gradient backgrounds are a staple of internet meme aesthetics and work well in `laser_cat_style`.

## Characteristics
- Smooth color transitions
- No texture or detail
- Synthetic feel
- High saturation
- Simple readability

## Use cases
- Single-gag hero mode
- Iconic compositions
- Sticker scatter support
- Diva cat scenes
- Graphic poster mode

## Best practices
- Strong color contrast with cat
- Rainbow or neon palettes
- Clean separation from foreground
- Meme-native energy

## Color families
- Rainbow gradient
- Sunset gradient
- Neon gradient
- Pastel gradient
- Black to color

## Drift risks
- Too subtle (loses meme energy)
- Too tasteful (becomes design)
- Wrong color temperature
