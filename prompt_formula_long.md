# Prompt Formula (Long)

A detailed, comprehensive prompt formula for `laser_cat_style`.

## Extended formula

```
[cat subject] + [pose/expression] + [crop/framing] + [absurd premise] + [objects] + [effects] + [background] + [lighting] + [color] + [artifacts] + [composition] + [tone] + [style cues]
```

## Full example

```
A blank-faced orange tabby cat in loaf pose, big head crop, seated on a giant cheeseburger as if it were a throne, surrounded by floating french fries and a milkshake, shooting rainbow laser beams from its eyes, set against a vibrant purple and pink nebula galaxy background with scattered stars, dramatic rim lighting with tacky neon glow, oversaturated candy colors with rainbow accents, visible cutout edges and slight JPEG compression artifacts, sticker scatter composition with the cat as central hero subject, absurd campy internet meme aesthetic, lowbrow cosmic collage style, correct badness, 2010s internet trash energy
```

## Component breakdown

### 1. Cat subject
- Breed/type: orange tabby, black cat, fluffy white
- Distinctive features: goblin face, wide eyes, smug expression

### 2. Pose/expression
- Pose: loaf, upright humanoid, sprawled, mid-jump
- Expression: blank, confused, smug, feral, screaming

### 3. Crop/framing
- Full body, half body, big head, face only, awkward crop

### 4. Absurd premise
- The core joke: riding, throne, merge, destroyer, witness

### 5. Objects
- Food: pizza, burger, fries, taco, bread
- Glam: sunglasses, diamonds, jewelry
- Sci-fi: UFOs, planets, stars
- Household: toilet paper, boxes
- Co-stars: sharks, dogs, unicorns

### 6. Effects
- Lasers: red, rainbow, divergent
- Emissions: rainbow puke, sparkle stream
- Glow: aura, halo, edge glow

### 7. Background
- Galaxy: nebula, spiral, star field
- Gradient: rainbow, neon, sunset
- Void: black stage
- Scene: city, beach, apocalypse

### 8. Lighting
- Rim lighting, dramatic, neon glow, tacky

### 9. Color
- Oversaturated, candy, neon, rainbow, synthetic

### 10. Artifacts
- Cutout edges, JPEG crust, glow bleed, correct badness

### 11. Composition
- Sticker scatter, single-gag hero, threat tableau

### 12. Tone
- Absurd, campy, stupid, internet-native

### 13. Style cues
- Meme collage, lowbrow, 2010s internet, tacky
