# Common Failure Modes

This file documents the most common ways `laser_cat_style` images fail or drift.

Understanding failure modes helps:
- diagnose weak outputs
- prevent drift
- improve prompts
- refine generation systems

---

# Core principle

Failure modes are not random.
They are predictable patterns of drift away from style identity.

Each failure mode has:
- symptoms
- causes
- fixes

---

# Failure Mode 1: Polished Drift

## Symptoms
- too refined
- too clean
- too designed
- seamless realism
- no visible collage
- no crust

## Causes
- over-correction
- taste overriding identity
- fear of "bad" execution

## Fixes
- add roughness
- add glow
- add artifact
- reduce realism
- increase tackiness
- reintroduce collage visibility

---

# Failure Mode 2: Generic Drift

## Symptoms
- "cat in space" with no specific logic
- lacks recognizable motifs
- no absurd context
- no object nonsense
- could be any cosmic cat

## Causes
- weak motif selection
- missing escalation
- no composition mode
- under-developed premise

## Fixes
- add specific motifs
- add absurd context
- add object nonsense
- strengthen composition mode
- activate humor engines

---

# Failure Mode 3: Serious Drift

## Symptoms
- genuine fantasy or sci-fi art
- dramatic without humor
- cat looks competent
- no stupidity
- cinematic grading

## Causes
- too much epic treatment
- loss of cat idiocy
- missing tone contradiction

## Fixes
- add cat idiocy
- add absurd context
- add tacky elements
- reduce cinematic drama
- reintroduce humor

# Diagnostic approach

When an image fails:

1. Identify symptoms
2. Match to failure mode
3. Identify causes
4. Apply fixes
5. Re-evaluate

---

# Bottom line

Common failure modes are predictable and fixable.

The main categories are:
- drift types (polished, generic, serious, cute, cool, random)
- execution failures (clutter, under-escalation, object dominance, effect overwhelm)
- visual failures (muddy colors, weak silhouette)
- communication failures (unclear premise, tone mismatch)

Understanding these modes allows:
- faster diagnosis
- targeted fixes
- drift prevention
- better outputs
