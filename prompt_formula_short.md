# Prompt Formula (Short)

A quick, reusable prompt formula for `laser_cat_style`.

## Basic formula

```
[cat description] + [absurd context] + [intensifier] + [background] + [style cue]
```

## Example

```
blank-faced loaf cat on giant pizza slice, rainbow laser eyes, galaxy background, internet meme collage style, tacky glow
```

## Components

### Cat description
- Pose: loaf, upright, sprawled, mid-action
- Expression: blank, confused, smug, goblin
- Visibility: clear silhouette, readable

### Absurd context
- Food: pizza, burger, sandwich, bread
- Objects: toilet paper, box, accessories
- Mounts: shark, unicorn, UFO

### Intensifier
- Lasers: red, rainbow, beam
- Emissions: rainbow puke, glow, aura
- Effects: sparkle, glitter, neon

### Background
- Galaxy: nebula, star field
- Gradient: rainbow, neon, sunset
- Void: black stage
- Scene: city, beach, apocalypse

### Style cue
- Internet meme collage
- Tacky glow
- Rough cutout
- Sticker scatter
- Correct badness

## Quick templates

### Food cat
```
[cat] on [food] in [background], [effect], tacky glow, meme style
```

### Laser cat
```
[cat] with [laser type] over [target], [background], internet aesthetic
```

### Giant cat
```
giant [cat] over [scene], [effect], fake epic, absurd scale
```

### Scatter cat
```
[cat] in [background] with [objects], sticker scatter, maximalist
```
