# Repair Strategies

This file provides specific strategies for repairing drifted or weak `laser_cat_style` images.

---

# Core principle

Repair requires:
1. Diagnosis of the problem
2. Selection of appropriate strategy
3. Application of targeted fixes
4. Re-evaluation

Not all problems require the same fix.
Not all fixes work for all problems.

---

# Strategy 1: Add Roughness

## Problem
Image is too polished, too clean, too refined.

## Solution
Introduce collage visibility and artifact.

## Tactics
- Add visible cutout edges
- Add glow or halo
- Add JPEG artifact simulation
- Reduce seamless integration
- Increase color clipping

## When to use
Polished drift, cool drift, refined drift

---

# Strategy 2: Add Absurd Context

## Problem
Image is generic "cat in space" with no specific joke.

## Solution
Introduce absurd premise and object nonsense.

## Tactics
- Add food items
- Add household objects
- Add glam accessories
- Add UFOs or sci-fi elements
- Create ridiculous scale relationships

## When to use
Generic drift, under-escalation, weak premise

---

# Strategy 3: Add Cat Idiocy

## Problem
Image is too serious, too competent, too heroic.

## Solution
Introduce stupidity and incompetence.

## Tactics
- Change cat expression to blank or confused
- Use loaf or passive pose
- Add absurd context that contradicts power
- Make cat look unqualified for treatment

## When to use
Serious drift, heroic drift, power without idiocy

---

# Strategy 4: Strengthen Composition Mode

## Problem
Image has no clear structure or hierarchy.

## Solution
Apply recognizable composition mode.

## Tactics
- For sticker scatter: add object cloud, distribute elements
- For single-gag hero: simplify, strengthen focal subject
- Establish clear hierarchy
- Improve visual rhythm

## When to use
Random drift, clutter collapse, unclear premise

---

# Strategy 5: Add Style-Native Motifs

## Problem
Image lacks recognizable style elements.

## Solution
Introduce core motif families.

## Tactics
- Add laser eyes or rainbow puke
- Add food items
- Add glam accessories
- Add UFOs
- Add cosmic debris

## When to use
Generic drift, weak motif selection, missing escalation

---

# Strategy 6: Adjust Tone

## Problem
Image has wrong emotional register.

## Solution
Shift tone toward absurd, campy, stupid.

## Tactics
- Add tacky elements
- Add goblin energy
- Reduce refinement
- Increase color violence
- Add internet-trash aesthetic

## When to use
Serious drift, cute drift, cool drift, wrong tone

---

# Strategy 7: Simplify

## Problem
Image is too cluttered, muddy, or noisy.

## Solution
Reduce elements, strengthen focal subject.

## Tactics
- Remove competing objects
- Increase cat size
- Reduce object count
- Improve contrast
- Clarify silhouette

## When to use
Clutter collapse, object dominance, effect overwhelm, muddy colors

---

# Strategy 8: Escalate

## Problem
Image is too simple, too plain, underwhelming.

## Solution
Increase visual intensity and absurdity.

## Tactics
- Add effects (lasers, glow, aura)
- Add more objects
- Increase scale absurdity
- Add background drama
- Push color saturation

## When to use
Under-escalation, weak premise, too simple

---

# Strategy 9: Rebalance Deity/Idiot

## Problem
Image has only one pole of the core tension.

## Solution
Add the missing pole.

## Tactics
- If too deity: add cat idiocy, blank expression, absurd context
- If too idiot: add dramatic treatment, cosmic background, visual power

## When to use
Imbalance, missing contradiction, weak humor engine

---

# Strategy 10: Improve Cat Visibility

## Problem
Cat is lost, obscured, or secondary.

## Solution
Strengthen cat presence and centrality.

## Tactics
- Increase cat size
- Position cat centrally
- Reduce competing elements
- Add glow or halo to cat
- Simplify around cat

## When to use
Object dominance, effect overwhelm, cat not central

---

# Strategy 11: Fix Color

## Problem
Colors are muddy, desaturated, or wrong.

## Solution
Adjust palette toward style-native colors.

## Tactics
- Increase saturation
- Add neon or rainbow elements
- Improve contrast
- Use synthetic palettes
- Reduce naturalism

## When to use
Muddy colors, desaturation, naturalistic drift

---

# Strategy 12: Clarify Premise

## Problem
Joke does not land, premise unclear.

## Solution
Strengthen the core absurdity.

## Tactics
- Make cat-object relation more obvious
- Simplify composition
- Add visual cues
- Reduce complexity
- Make the "what the fuck" immediate

## When to use
Unclear premise, weak humor engine, no immediate read

---

# Strategy selection guide

| Problem | Primary Strategy | Secondary Strategy |
|---|---|---|
| Too polished | Add roughness | Adjust tone |
| Generic | Add absurd context | Add style-native motifs |
| Too serious | Add cat idiocy | Adjust tone |
| No structure | Strengthen composition | Simplify |
| Weak motifs | Add style-native motifs | Escalate |
| Wrong tone | Adjust tone | Add absurd context |
| Too cluttered | Simplify | Improve cat visibility |
| Too simple | Escalate | Add absurd context |
| Imbalanced | Rebalance deity/idiot | Adjust treatment |
| Cat lost | Improve cat visibility | Simplify |
| Muddy colors | Fix color | Escalate |
| Unclear premise | Clarify premise | Strengthen composition |

---

# Repair workflow

1. **Diagnose**
   - Run diagnostic checklist
   - Identify failure mode
   - Determine primary problem

2. **Select strategy**
   - Match problem to strategy
   - Consider secondary strategies
   - Plan specific tactics

3. **Apply fixes**
   - Implement selected tactics
   - Make targeted changes
   - Preserve what works

4. **Re-evaluate**
   - Run checklist again
   - Compare to original
   - Determine if additional fixes needed

5. **Iterate**
   - Repeat as necessary
   - Stop when style identity is clear

---

# Bottom line

Repair is systematic, not random.

Each problem has appropriate strategies.
Each strategy has specific tactics.

The goal is not perfection.
The goal is **style identity restoration**.

A repaired image should be:
- recognizably `laser_cat_style`
- better than the drifted version
- functional and funny
