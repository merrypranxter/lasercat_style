# Diagnostic Checklist

This file provides a systematic checklist for evaluating whether an image belongs to `laser_cat_style` and identifying any drift.

---

# Core principle

Diagnosis requires systematic evaluation across multiple dimensions.

A single weak element may not cause drift.
Multiple weak elements probably do.

---

# Section 1: Cat Centrality

## Check 1.1: Is a cat the main subject?
- [ ] Yes, clearly central
- [ ] Yes, but competing elements
- [ ] No, cat is secondary
- [ ] No cat present

**Pass threshold:** Yes, clearly central or yes with minor competition

## Check 1.2: Is the cat readable?
- [ ] Clear silhouette
- [ ] Readable but not iconic
- [ ] Muddy or unclear
- [ ] Lost in effects

**Pass threshold:** Clear silhouette or readable

## Check 1.3: Does the cat have expression?
- [ ] Clear readable expression
- [ ] Some expression visible
- [ ] Neutral/blank
- [ ] Obscured by effects

**Pass threshold:** Any readable expression

## Check 1.4: Is the cat both powerful and stupid?
- [ ] Yes, clear deity/idiot balance
- [ ] Leans one way but both present
- [ ] Only one pole present
- [ ] Neither pole present

**Pass threshold:** Both poles present, even if unbalanced

---

# Section 2: Collage Logic

## Check 2.1: Does the image feel assembled?
- [ ] Clearly composited
- [ ] Semi-assembled
- [ ] Seamless but synthetic
- [ ] Fully realistic

**Pass threshold:** Clearly or semi-assembled

## Check 2.2: Is there visible cutout or masking?
- [ ] Yes, obvious
- [ ] Some visible edges
- [ ] Subtle only
- [ ] None visible

**Pass threshold:** Any visible evidence

## Check 2.3: Is there source mismatch?
- [ ] Clear lighting/resolution mismatch
- [ ] Some mismatch
- [ ] Subtle only
- [ ] Fully consistent

**Pass threshold:** Any mismatch acceptable

## Check 2.4: Is there preserved "wrongness"?
- [ ] Clear roughness/glow/artifact
- [ ] Some roughness
- [ ] Very subtle
- [ ] Polished clean

**Pass threshold:** Any roughness or glow

---

# Section 3: Motifs

## Check 3.1: Are motifs from correct families?
- [ ] Yes, multiple style-native motifs
- [ ] Yes, one or two
- [ ] Partial overlap
- [ ] No style-native motifs

**Pass threshold:** At least one style-native motif

## Check 3.2: Are motifs appropriately used?
- [ ] Integrated into scene logic
- [ ] Present but not integrated
- [ ] Random placement
- [ ] Absent

**Pass threshold:** Present and integrated

## Check 3.3: Is there visual escalation?
- [ ] Multiple intensifiers
- [ ] One clear intensifier
- [ ] Weak escalation
- [ ] No escalation

**Pass threshold:** At least one clear intensifier

---

# Section 4: Composition

## Check 4.1: Is the composition mode recognizable?
- [ ] Clear sticker scatter
- [ ] Clear single-gag hero
- [ ] Ambiguous but structured
- [ ] No clear mode

**Pass threshold:** Recognizable mode or clear structure

## Check 4.2: Is there clear hierarchy?
- [ ] Clear primary subject
- [ ] Primary with competition
- [ ] Flat hierarchy
- [ ] No hierarchy

**Pass threshold:** Clear or manageable hierarchy

## Check 4.3: Is the scene readable?
- [ ] Instant read
- [ ] Quick read
- [ ] Requires study
- [ ] Unreadable

**Pass threshold:** Instant or quick read

---

# Section 5: Tone

## Check 5.1: Is the tone absurd?
- [ ] Clearly absurd
- [ ] Somewhat absurd
- [ ] Ambiguous
- [ ] Serious

**Pass threshold:** Clearly or somewhat absurd

## Check 5.2: Is the tone campy or tacky?
- [ ] Clearly campy/tacky
- [ ] Some camp
- [ ] Neutral
- [ ] Refined

**Pass threshold:** Any camp or tackiness

## Check 5.3: Is the tone internet-native?
- [ ] Clearly meme-native
- [ ] Somewhat internet
- [ ] Ambiguous
- [ ] Traditional

**Pass threshold:** Meme-native or internet-adjacent

---

# Section 6: Color and Effects

## Check 6.1: Is the palette synthetic/bright?
- [ ] Clearly synthetic
- [ ] Somewhat synthetic
- [ ] Natural with accents
- [ ] Fully natural

**Pass threshold:** Synthetic or synthetic-adjacent

## Check 6.2: Are effects present?
- [ ] Multiple effects
- [ ] One clear effect
- [ ] Subtle effects
- [ ] No effects

**Pass threshold:** At least one clear effect

## Check 6.3: Is there glow or halo?
- [ ] Clear glow/halo
- [ ] Subtle glow
- [ ] No glow
- [ ] Natural lighting

**Pass threshold:** Any glow acceptable

---

# Scoring

## Strong pass (core)
- All sections pass
- Most checks at "yes, clearly" or "multiple"
- No critical failures

## Weak pass (adjacent)
- Most sections pass
- Some checks at minimum threshold
- One or two minor failures

## Weak fail (peripheral)
- Some sections fail
- Multiple checks at minimum or below
- Clear drift signs

## Strong fail (not this style)
- Multiple sections fail
- Critical elements missing
- Belongs to different style

---

# Drift diagnosis

## Polished drift
- Section 2: low scores
- Section 6: low effect scores
- Tone: too refined

## Generic drift
- Section 3: no style-native motifs
- Section 4: no clear mode
- Section 5: weak absurdity

## Serious drift
- Section 1: no idiocy
- Section 5: too serious
- Tone: wrong register

## Cute drift
- Section 1: too cute
- Section 5: not absurd
- Tone: kawaii

## Cool drift
- Section 5: too cool
- Section 6: too refined
- Tone: vaporwave

## Random drift
- Section 4: no clear mode
- Section 3: no clear motifs
- Overall: no structure

---

# Bottom line

Systematic diagnosis prevents drift.

Use this checklist to:
- evaluate outputs
- identify weaknesses
- apply targeted fixes
- maintain style identity

A passing image is not necessarily perfect.
It is recognizably `laser_cat_style`.
