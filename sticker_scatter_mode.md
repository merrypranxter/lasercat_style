# Sticker Scatter Mode

Sticker scatter mode is one of the two primary composition modes in `laser_cat_style`.

It creates dense, modular, object-cloud compositions where multiple discrete elements float across a field like a curated desktop of cursed PNGs.

This file defines how sticker scatter mode works.

---

# Core principle

In sticker scatter mode:
- multiple objects float independently
- the background acts as stage, not environment
- object equality is high
- nonsense density is high
- the scene reads like a sticker cloud
- physical realism is minimal

The joy comes from the accumulation, not from a single clear gag.

---

# Why sticker scatter works

Sticker scatter is powerful because it:
- maximizes visual richness
- supports procedural generation
- creates meme-native chaos
- allows object variety
- feels internet-collage authentic
- is highly reusable

A cloud of floating junk with a cat in the middle is immediately `laser_cat_style`.

# Composition structure

## Layer 1: Background field
Galaxy, gradient, void, or cosmic stage.

## Layer 2: Main cat(s)
Central focal subject(s).

## Layer 3: Large objects
Food, vehicles, major accessories.

## Layer 4: Medium objects
Fries, diamonds, smaller food, accessories.

## Layer 5: Small objects
Sparkles, particles, tiny debris.

## Layer 6: Effects
Glows, halos, overlays.

---

# Object placement rules

## Distribute across the frame
Avoid clustering everything in one zone.

## Vary sizes
Mix large, medium, and small objects.

## Create visual rhythm
Objects should lead the eye around the scene.

## Preserve cat centrality
The cat must remain the main focal point.

## Allow overlap
Some object overlap adds depth and density.

## Avoid uniform spacing
Randomness feels more authentic than grid alignment.

---

# Common failure modes

## Too cluttered
Objects overwhelm the cat.

Fix:
- reduce object count
- increase cat size
- simplify the scatter
- improve focal hierarchy

## Too sparse
The scatter feels thin.

Fix:
- add more objects
- increase object variety
- add smaller debris
- add effects

## No cat centrality
The cat is lost in the cloud.

Fix:
- make the cat larger
- position the cat centrally
- reduce competing large objects
- add glow or halo to cat

## Uniform spacing
Objects feel arranged, not scattered.

Fix:
- add randomness
- vary distances
- allow clustering
- break grid patterns

## Muddy colors
Objects blend together.

Fix:
- increase color contrast
- reduce object overlap
- simplify palette
- add brighter accents
