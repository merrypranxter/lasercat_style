# Cat Power Roles

In `laser_cat_style`, the cat does not merely appear in the image.
It occupies a role.

These roles determine how the cat behaves visually, compositionally, and tonally.

This file defines the main power-role archetypes used throughout the style.

---

# Core principle

A cat can function as:
- god
- idiot
- destroyer
- witness
- rider
- throne-occupant
- object-inhabitant
- emission-source
- icon
- participant in chaos

These roles help transform the style from random cat collage into a usable visual system.

---

# Why roles matter

Role assignment helps define:
- scene focus
- scale logic
- tone
- composition mode
- object relation
- prompt intent
- drift diagnosis

Without roles, the repo would only know that there is "a cat."
With roles, the repo can understand what the cat is **doing** in the visual logic.

---

# Primary roles

## 1. Cat as Deity

### Definition
The cat appears powerful, central, elevated, or fake-divine.

### Common signs
- oversized scale
- throne posture
- beam output
- central framing
- looming presence
- object orbit
- elevated placement
- visual calm amid chaos

### Typical branches
- monster/god-cat
- space collage cat
- graphic collage cat

### Emotional effect
- majestic stupidity
- divine nonsense
- absurd authority

### Risks
- drifting into serious fantasy or kaiju art
- losing the humor

---

## 2. Cat as Idiot

### Definition
The cat appears gloriously unqualified, confused, blank, or simply too dumb for the circumstances.

### Common signs
- blank face
- awkward pose
- accidental comedy
- passive expression inside ridiculous context
- object occupation without visible awareness

### Typical branches
- cosmic food cat
- household object cat
- witness scenes
- many hero-gag images

### Emotional effect
- affection
- hilarity
- chaotic innocence

### Risks
- becoming too passive or compositionally weak
- collapsing into generic funny-cat imagery

---

## 3. Cat as Destroyer

### Definition
The cat is the active source of damage, force, or absurd visual aggression.

### Common signs
- laser eyes
- rainbow attack
- giant threatening scale
- open-mouth emission
- city destruction context
- dominant forward energy

### Typical branches
- laser cat
- monster/god-cat
- beach/city havoc scenes

### Emotional effect
- fake terror
- destructive camp
- comic power fantasy

### Risks
- drifting into action-poster seriousness
- becoming too violent without enough stupidity

---

## 4. Cat as Witness

### Definition
The cat observes or exists within the nonsense rather than directly causing it.

### Common signs
- side glance
- calm loaf amid absurdity
- small secondary placement
- expression of mild concern or blank neutrality
- passive relation to floating objects or cosmic event

### Typical branches
- space collage cat
- co-star scenes
- sticker-scatter mode
- graphic collage branch

### Emotional effect
- deadpan humor
- low-energy absurdity
- accidental philosophical cat presence

### Risks
- losing cat centrality
- turning into filler

---

## 5. Cat as Rider

### Definition
The cat rides another object, animal, or absurd platform.

### Common signs
- pizza ride
- shark ride
- unicorn ride
- taco ride
- floating food or creature mount
- action posture or perched stability

### Typical branches
- cosmic food cat

- hero-gag mode

### Emotional effect

## 6. Cat as Throne-Occupant

### Definition
The cat sits on an absurd object as if it deserves to own it.

### Common signs
- loaf posture
- smug calm
- object used like couch, raft, or royal seat
- strong central stillness
- cat above or atop nonsense without concern

### Typical branches
- sandwich cat
- pizza cat
- object cat
- deity-adjacent food cat

### Emotional effect
- entitlement
- regal stupidity
- deadpan absurdity

### Risks
- scene becoming visually static if not supported by background or contrast

---

## 7. Cat as Emission Source

### Definition
The cat is primarily important because its eyes or mouth are generating the image's main effect.

### Common signs
- eye lasers
- rainbow tears
- rainbow puke
- glowing eyes
- mouth beam
- aura source logic

### Typical branches
- laser / psychedelic / drug cat
- destroyer cat
- graphic poster cat

### Emotional effect
- intensity
- absurd power
- meme escalation

### Risks
- effect overpowering the cat face
- becoming generic laser-eye meme

---

## 8. Cat as Object-Inhabitant

### Definition
The cat is embedded inside, framed by, or merged with an object.

### Common signs
- cat in bread
- cat wrapped in toilet paper
- cat fused with object silhouette
- object functioning like costume or habitat

### Typical branches
- household object cat
- cosmic food cat
- hero-gag mode

### Emotional effect
- blunt absurdity
- delight through literalness
- visual compression of joke

### Risks
- becoming too gimmicky without strong surrounding context

---

## 9. Cat as Icon

### Definition
The cat acts like a symbolic sticker, emblem, or repeated motif rather than an individual character.

### Common signs
- small repeated cats
- simplified or emblematic cat crops
- cat functioning as object among objects
- chorus use in group compositions

### Typical branches
- sticker scatter mode
- group chaos mode
- wallpaper-like meme fields

### Emotional effect
- pattern humor
- collectible nonsense
- visual rhythm

### Risks
- cat losing distinct presence
- over-flattening into generic motif spam

---

## 10. Cat as Participant in Shared Chaos

### Definition
The cat is one actor among multiple absurd elements rather than the sole source or symbol.

### Common signs
- cat + UFO + fries + other animal + diamond
- cat sharing stage with multiple nonsense agents
- relational chaos

### Typical branches
- space collage cat
- co-star creature cat
- glam trash cat
- sticker scatter mode

### Emotional effect
- ensemble absurdity
- cosmic social nonsense
- chaotic scene richness

### Risks
- cat role becoming ambiguous
- overall focal hierarchy collapsing

# Role combinations

Strong examples often combine roles.

## Deity + Destroyer
Giant laser cat over city.

## Idiot + Throne-Occupant
Loaf cat on sandwich.

## Rider + Destroyer
Cowboy cat riding shark while rainbow erupts.

## Witness + Icon
Small cat in object cloud composition.

## Emission Source + Deity
Central cat face with overwhelming beam output.

## Object-Inhabitant + Idiot
Cat framed by bread, staring blankly into the void.

Role overlap is not a problem.
It is often where the strongest images live.

---

# Role selection guidelines

Ask:
- Is the cat meant to rule the image?
- Is the cat the punchline?
- Is the cat the force?
- Is the cat reacting or simply existing?
- Is the cat mounted, fused, or enthroned?
- Is the cat the thing generating the effect?

Choose one primary role and optionally one secondary role.
That is usually enough to structure the scene.

---

# Anti-drift role warnings

## Too much deity, not enough cat stupidity
Image becomes fantasy or sci-fi poster.

## Too much idiot, not enough compositional force
Image becomes ordinary funny cat meme.

## Too much destroyer, not enough absurdity
Image becomes action art.

## Too much icon use, not enough central hierarchy
Image becomes clutter wallpaper.

## Too much participant role without role clarity
Image loses memorable logic.

---

# Bottom line

The cat in `laser_cat_style` is not only a subject.
It is a role-bearing visual engine.

The most important role tensions are:
- deity vs idiot
- destroyer vs witness
- rider vs throne-occupant
- emission source vs passive icon

Those tensions are where the style gets its power.

The cat must always be doing more than merely being there.
