# Substyle Branches

`laser_cat_style` is not one single locked image formula.
It is a family of related branches that share core DNA while emphasizing different motifs, tones, and scene structures.

This file defines the major branches.

These branches are not isolated silos.
They overlap freely.
A strong image may belong to two or three at once.

---

# Why branch the style?

Without substyle branches, the repo risks flattening everything into:
- "cats in space"
- "cats with lasers"
- "random cat meme collage"

That would be useless.

Branching helps:
- classify examples
- build stronger prompts
- define scene archetypes
- support tagging and app logic
- prevent drift into generic slop

# 1. Cosmic Food Cat

## Definition
Cats fused with, riding, sitting on, emerging from, or juxtaposed against food items—especially junk food or simple meme-legible foods.

Common examples:
- cat on pizza
- cat on sandwich
- cat floating with burger, fries, and milkshake
- cat wrapped in bread
- cat on taco
- food as throne, mount, raft, or halo

## Visual priorities
- food readability
- absurd pairing clarity
- flat or sticker-like separation
- bright or cosmic background support
- immediate joke legibility

## Typical tone
- dumb
- cute
- iconic
- low-stakes surreal
- novelty-poster adjacent

## Frequent motifs
- pizza slices
- burgers
- fries
- tacos
- sandwiches
- milkshakes
- floating snack clouds

## Common drift
- becoming simply "cute cat with food"
- becoming polished food illustration
- losing cosmic or collage escalation

- altered-state color treatment

## Frequent motifs

- psychedelic irises
- glow halos

# 4. Monster / God Cat

## Definition
Cats scaled up into destructive, divine, ominous, or absurdly authoritative beings.

Common examples:
- giant cat over a city
- city-destroying laser cat
- kaiju cat
- blank-eyed beam cat looming over buildings
- cat as cosmic ruler
- cat as judgment machine

## Visual priorities
- dramatic scale mismatch
- dominance composition
- skyline or scene vulnerability
- power signaling
- absurd overstatement

## Typical tone
- majestic
- threatening
- funny
- apocalyptic in a stupid way
- fake mythic

## Frequent motifs
- cityscapes
- clouds
- black-and-white urban scenes
- giant silhouette cat
- beams destroying architecture
- authority pose
- blank or glowing eyes

## Common drift
- becoming serious kaiju art
- becoming sleek sci-fi poster design
- losing the cat's idiot essence

# 5. Glam Trash Cat

## Definition
Cats enhanced with tacky glam or lowbrow luxury signifiers.

Common examples:
- rhinestone sunglasses
- diamonds floating nearby
- bikini cats
- bright accessory overload
- "diva" cat in cosmic scene

## Role in the system
This is usually a cross-branch intensifier rather than a fully separate branch.

# 6. Household Object Cat

## Definition
Cats merged with ordinary objects that become absurd costume, wrapper, or environment elements.

Common examples:
- cat in bread
- cat wrapped in toilet paper
- cat inside object frame
- cat partially replaced by domestic nonsense

## Role in the system
Often overlaps strongly with:
- Food Cat

## Common drift

- losing compositional escalation

# 7. Co-Star Creature Cat

## Definition
Cats paired with other animals as co-riders, enemies, witnesses, mounts, or absurd scene partners.

Common examples:
- cat riding shark
- cat on alpaca or unicorn
- cat paired with another strange beast
- cat + creature joint tableau

## Role in the system
This branch expands narrative and chaos possibilities.
It often overlaps with:
- Food Cat
- Space Collage Cat
- Monster Cat

## Common drift
- becoming generic absurd-animal image
- co-star overpowering cat centrality

---

# 8. Graphic Collage Cat

## Definition
A more graphic, poster-like, or Tumblr-adjacent branch with stronger contrast, less clutter, and more stylized collage framing.

Common examples:
- black-and-white city with rainbow beams
- grayscale cat with one or two bright insertions
- cleaner but still obviously composited layouts

## Visual priorities
- silhouette clarity
- graphic contrast
- collage symbolism
- reduced clutter
- strong central focal device

## Role in the system
This is an allowed edge branch.
It belongs when the image still retains:
- cat centrality
- collage obviousness
- absurd escalation
- style-correct energy

## Common drift
- becoming generic Tumblr surreal collage
- becoming too cool, too tasteful, or too fashion-oriented

# Branch combinations

Many strong images combine multiple branches.

Examples:

## Food + Space Collage
Cat on pizza in galaxy with extra floating food.

## Psychedelic + Glam Trash
Cat with glowing eyes, sunglasses, diamonds, and a cosmic void.

## Monster + Graphic Collage
Giant rainbow-beam cat over grayscale city.

## Household Object + Hero Gag
Cat framed by bread with a dramatic space backdrop.

## Co-Star Creature + Psychedelic
Cat riding shark while rainbow puke or lasers intensify the scene.

Branch overlap is normal and should often be encouraged.

- Monster / God Cat

## Secondary branches
These usually function as modifiers or intensifiers:
- Glam Trash Cat
- Household Object Cat
- Co-Star Creature Cat
- Graphic Collage Cat

---

# Classification guidance

When assigning a branch:
- choose the **dominant gag engine** first
- choose **secondary intensifiers** second
- do not over-tag every possible overlap
- preserve the image's main visual logic

Good classification:
- primary branch
- secondary branch
- optional intensifier tags

---

# Bottom line

The substyle branches prove that `laser_cat_style` is not a single joke repeated forever.

It is a modular family of related image behaviors held together by:
- cats
- collage
- absurdity
- escalation
- internet-native wrongness

The branches help the repo preserve that structure without flattening everything into rainbow-cat soup.
