# Humor Engines

`laser_cat_style` is funny, but the humor does not come from randomness.
It comes from specific, repeatable engines.

This file defines the main humor mechanisms that make the style work.

---

# Core principle

The style's humor is structural, not accidental.

It relies on:
- juxtaposition
- scale absurdity
- power mismatch
- context wrongness
- object nonsense
- visual escalation
- cat idiocy

Understanding these engines helps:
- build better prompts
- diagnose weak images
- prevent drift
- generate variations

---

# 1. Juxtaposition Engine

## How it works
Two things that do not belong together are forced together.

## Examples
- cat + pizza + galaxy
- cat + laser + city
- cat + bread + cosmic void
- cat + glam accessories + space

## Why it works
The brain recognizes the mismatch and processes it as humor.

## Strengths
- immediate
- universal
- highly reusable

## Risks
- can become generic if the juxtaposition is too common

---

# 2. Scale Absurdity Engine

## How it works
Size relationships are obviously wrong.

## Examples
- cat larger than city
- pizza large enough to ride
- tiny cat, giant milkshake
- kaiju cat over skyline

## Why it works
Scale is a fundamental cognitive category. Violating it is instantly funny.

## Strengths
- iconic
- memorable
- highly visual

## Risks
- timid wrongness is boring
- must be bold and obvious

---

# 3. Power Mismatch Engine

## How it works
The cat has power it does not deserve or understand.

## Examples
- blank-faced cat with laser eyes
- loaf cat destroying city
- idiot cat as cosmic deity
- confused cat on throne

## Why it works
The contradiction between appearance and capability is inherently comic.

## Strengths
- character-based
- emotionally engaging
- highly meme-native

## Risks
- cat must remain readable as incompetent

---

# 4. Context Wrongness Engine

## How it works
The cat is placed in a context that makes no sense for a cat.

## Examples
- cat in space
- cat on shark
- cat in bread
- cat with UFO
- cat in apocalypse

## Why it works
Cats are domestic animals. Cosmic or dangerous contexts are absurd.

## Strengths
- highly relatable
- easy to generate variations
- strong visual contrast

## Risks
- can become generic "cat in weird place" without additional escalation

---

# 5. Object Nonsense Engine

## How it works
Random objects accumulate around the cat in absurd density.

## Examples
- cat + fries + diamond + milkshake + UFO
- object cloud scenes
- sticker scatter mode
- floating junk debris

## Why it works
The brain tries to find pattern in chaos. The failure to find pattern is funny.

## Strengths
- highly style-native
- supports procedural generation
- creates visual richness

## Risks
- too much clutter kills focal hierarchy
- objects must remain readable

---

# 6. Visual Escalation Engine

## How it works
The image keeps adding intensity until it becomes ridiculous.

## Examples
- cat → laser eyes → galaxy background → city destruction → rainbow beams
- cat → on pizza → in space → with fries → with diamonds → with glow

## Why it works
Excess becomes comedy. More is more until it is stupid.

## Strengths
- highly flexible
- supports variation
- creates spectacle

## Risks
- can become visual noise if not controlled
- must preserve cat centrality

---

# 7. Cat Idiocy Engine

## How it works
The cat's expression or behavior undermines any attempt at grandeur.

## Examples
- blank stare during apocalypse
- loaf posture on absurd throne
- confused face with cosmic power
- goblin scream with rainbow puke

## Why it works
The cat is the straight man in its own absurd scene.

## Strengths
- character-based
- emotionally engaging
- highly meme-native

## Risks
- cat must remain visible and expressive

---

# Engine combinations

Strong images often combine multiple engines.

## Example 1: Giant laser cat over city
- scale absurdity
- power mismatch
- context wrongness
- visual escalation

## Example 2: Cat on pizza in galaxy with fries
- juxtaposition
- scale absurdity
- context wrongness
- object nonsense

## Example 3: Blank-faced god-cat with rainbow puke
- power mismatch
- cat idiocy
- visual escalation

## Example 4: Cat in bread with diamond scatter
- juxtaposition
- object nonsense
- context wrongness

---

# Engine selection for prompts

When building a prompt, choose 2-3 engines:

## For fake epic
- scale absurdity
- power mismatch
- visual escalation

## For dumb delight
- juxtaposition
- object nonsense
- cat idiocy

## For cosmic nonsense
- context wrongness
- object nonsense
- visual escalation

## For tacky joy
- juxtaposition
- object nonsense
- visual escalation

---

# Engine failure diagnosis

## Image feels flat
Likely missing: scale absurdity or visual escalation

## Image feels random but not funny
Likely missing: cat idiocy or power mismatch

## Image feels generic
Likely missing: strong juxtaposition or object nonsense

## Image feels too serious
Likely missing: cat idiocy or context wrongness

---

# Bottom line

`laser_cat_style` humor is not random.
It is built from repeatable engines:

- juxtaposition
- scale absurdity
- power mismatch
- context wrongness
- object nonsense
- visual escalation
- cat idiocy

A strong image activates 2-3 of these engines.
A weak image activates none or only one weakly.

The engines are the load-bearing structure of the humor.
