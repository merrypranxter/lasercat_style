# Scene Archetypes

`laser_cat_style` uses a limited but powerful set of recurring scene structures.

These archetypes are not rigid templates.
They are recognizable patterns that help classify, generate, and understand the style.

This file defines the main scene archetypes.

---

# Core principle

Scene archetypes are reusable visual formulas that:
- deliver specific emotional effects
- use specific motif combinations
- follow specific composition logics
- create recognizable style moments

Understanding archetypes helps:
- build better prompts
- classify examples
- generate variations
- prevent drift

---

# Archetype 1: The Cosmic Throne

## Definition
A cat sits on an absurd object in a cosmic setting.

## Elements
- loaf or seated cat
- food or object throne
- galaxy or void background
- optional glow or aura
- optional scattered debris

## Tone
- entitlement
- absurd majesty
- passive power

## Branches
- cosmic food cat
- household object cat
- glam trash cat

## Example
Cat on burger in nebula with floating fries.

---

# Archetype combinations

Strong images often combine archetypes.

## Example 1: Giant laser cat over city
- kaiju threat
- beam judgment

## Example 2: Cat on pizza with fries scatter
- cosmic throne
- object cloud

## Example 3: Cat riding shark with rainbow trail
- ride
- beam judgment (variant)

## Example 4: Posterized cat face with glow
- freakout
- diva (if glam elements added)

---

# Archetype selection for prompts

When building a prompt, choose 1-2 archetypes:

## For fake epic
- beam judgment
- kaiju threat

## For dumb delight
- cosmic throne
- merge
- ride

## For cosmic chaos
- object cloud
- contact
- witness

## For visual overload
- freakout
- object cloud

## For ironic status
- diva
- cosmic throne

---

# Bottom line

Scene archetypes are reusable patterns that:
- deliver specific effects
- use specific combinations
- follow specific logics
- create recognizable moments

The ten core archetypes are:
1. cosmic throne
2. beam judgment
3. object cloud
4. ride
5. merge
6. kaiju threat
7. contact
8. freakout
9. witness
10. diva

A strong image often combines 2 archetypes.
A weak image activates none clearly.

Archetypes are the style's scene vocabulary.
