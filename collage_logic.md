# Collage Logic

`laser_cat_style` is fundamentally a collage style.
Even when an image is relatively clean, the visual logic should still feel assembled, composited, layered, or dragged together from mismatched pieces.

The collage should not disappear completely into seamless realism.

This file defines how collage works in the style.

---

# Core principle

The image should feel **built**, not naturally photographed.

That can mean:
- obvious cutouts
- visible source mismatch
- floating sticker logic
- inconsistent lighting
- inconsistent scale
- blunt layering
- artifact halos
- reused or isolated object fragments
- compositions that behave more like digital scrapbooking than real space

This built quality is a core part of the aesthetic.

---

# What collage means here

In `laser_cat_style`, collage usually implies one or more of the following:
- a cat isolated from its original source
- a background unrelated to the cat's original environment
- additional objects inserted with minimal realism correction
- visible seams, halos, or tonal mismatch
- a composition structured through placement rather than physically coherent space
- a scene that feels draggable, stackable, and modular

The collage may be:
- bad
- semi-bad
- medium competent
- surprisingly polished but still clearly synthetic

It should rarely become fully invisible.

---

# Core collage behaviors

## 1. Sticker logic

A very common behavior.
Objects feel like stickers placed on top of a background:
- cat
- burger
- UFO
- fries
- diamond
- pizza
- shark
- milkshake
- random accessory

Each element remains somewhat discrete.

This helps:
- readability
- humor clarity
- object equality
- drag-and-drop absurdity

Sticker logic is especially strong in scatter-mode compositions.

---

## 2. Cutout isolation

The cat and other objects are often clearly cut from somewhere else.

This can show up as:
- hard edge clipping
- fuzzy edge leftovers
- white halo
- soft mask blur
- background contamination
- shape simplification
- edge crunch from compression

These are not automatically flaws.
Often they are part of the correct look.

---

## 3. Source mismatch

The image may combine elements with very different:
- lighting
- resolution
- contrast
- sharpness
- color temperature
- photographic style
- era cues

This mismatch is often part of the joke.

Examples:
- soft cat + ultra-crisp burger
- blurry galaxy + sharp food PNG
- grayscale city + rainbow beam overlay
- neon glow on a cat lit by normal daylight

Correct mismatch feels like visual shamelessness.

---

## 4. Reality disrespect

The collage should not care too much about:
- consistent perspective
- coherent physics
- correct cast shadows
- natural gravity
- optical realism
- background-object integration

This does not mean unreadable chaos.
It means the scene is governed by comedic placement, not realism.

---

## 5. Layer hierarchy

Even chaotic images usually have a placement order.

A common order:
1. background field
2. main cat
3. large co-objects or vehicles
4. emission effects
5. scattered accent objects
6. glows / halos / overlays

This hierarchy helps the image remain legible even when stupid.

# Collage assembly models

## Model A: Hero Cutout

One main cat dominates the composition and everything else supports it.

Typical traits:
- oversized focal subject
- fewer total objects
- stronger silhouette clarity
- scene reads in one glance

Good for:
- giant laser cat
- sandwich cat
- shark rider cat
- acid-face cat

---

## Model B: Sticker Scatter

Many separate objects float across the frame like a cursed desktop.

Typical traits:
- multiple small inserts
- less realistic space
- higher nonsense density
- background acts as stage, not environment

Good for:
- fries + burger + milkshake cat scenes
- UFO + diamond + sunglass cat compositions
- multi-cat nebula chaos

---

## Model C: Object Merge

The cat is merged with one primary object.

Examples:
- cat in bread
- cat on sandwich
- cat wrapped in toilet paper
- cat as food mount occupant

The object and cat become a single unit.
This model benefits from very clear visual reading.

---

## Model D: Threat Tableau

The cat is placed into a bigger dramatic scene:
- city
- beach
- skyline
- apocalypse
- grayscale urban collage

The cat acts as force or event rather than just sticker content.

# Good collage wrongness

The style benefits from specific types of wrongness.

## Useful wrongness
- edge halo
- mismatched sharpness
- wrong scale
- arbitrary float placement
- too-bright glow
- obvious background replacement
- slightly rough selection
- repeated resave softness
- absurd object layering

These reinforce internet-native energy.

## Harmful wrongness
- unreadable silhouette
- object clutter so dense the gag disappears
- accidental visual mud
- edge contamination so bad the subject dissolves
- realism corrections that are half-done and awkward in a dead way
- effects that obscure the cat's face too much

The wrongness must remain functional.

---

# Compositional placement logic

Collage placement should obey humor before realism.

Questions to ask:
- what is the joke?
- what is the main focal subject?
- what needs to be seen first?
- what can remain secondary?
- what object is funniest when oversized?
- what object is funniest when small and floating?
- should the scene feel like one event or a cluster of tiny insults?

In this style, placement is meaning.

---

# Cutout quality tiers

## Tier 1: Crude cutout

Very rough edge, little cleanup, meme-native.

Use for:
- chaotic shitpost energy
- strong internet-trash authenticity
- images that benefit from bluntness

## Tier 2: Semi-clean cutout

Readable edges with some visible halo or softness.

Use for:
- most core examples
- balanced humor + legibility
- broad compatibility across branches

## Tier 3: Clean cutout with intentionally preserved synthetic feel

Technically better, but still not integrated like realism.

Use for:
- graphic collage branch
- more poster-like variants
- cleaner branch explorations

Tier 3 must not become seamless realism.

---

# Background relation strategies

## Background as stage
The background simply holds the cutouts.
It is not expected to physically contain them.

## Background as contrast field
The background exists to make the cat or emissions pop.

## Background as drama amplifier
The cat becomes funnier because the background is too epic.

## Background as ironic mismatch
The background adds a bizarre emotional or genre contradiction.

---

# Repetition and modularity

A major strength of collage logic is modular recombination.

The style can reuse:
- same cat, new object
- same background, new cat
- same beam logic, new pose
- same food set, different cat scale
- same object cloud, different hero image

This makes it ideal for:
- prompt systems
- generative art
- app logic
- meme variation engines

The style is not only collage-looking.
It is structurally collage-modular.

---

# Anti-drift collage rules

To remain core `laser_cat_style`, collage should usually preserve:
- obvious assembly logic
- cat centrality
- readable separation between elements
- some amount of visual incorrectness
- joke legibility over realism coherence

Avoid:
- seamless photobashing
- cinematic realism
- elegant surreal integration
- polished compositing that erases all crust
- hyper-consistent grading that turns everything into poster art

---

# Bottom line

The collage in `laser_cat_style` is not a technical limitation.
It is a style behavior.

The image should feel:
- assembled
- shameless
- modular
- visually wrong in useful ways
- more concerned with punchline and icon power than physical truth

In short:

the collage should hold together
but it should never behave too well
