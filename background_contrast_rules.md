# Background Contrast Rules

Background choice affects how well the cat and objects read. This file defines contrast principles.

## Core principle
The background should support foreground readability, not compete with it.

## Contrast types

### Value contrast
- Light cat on dark background
- Dark cat on light background
- Silhouette-friendly

### Color contrast
- Complementary colors
- Synthetic vs natural
- Neon vs muted

### Saturation contrast
- Saturated foreground
- Muted or gradient background
- Avoid equal saturation

### Detail contrast
- Detailed foreground
- Simple background
- Or vice versa for specific effects

## Background-cat pairings

### Dark-furred cats
- Galaxy backgrounds (stars provide points)
- Gradient backgrounds (color pop)
- Avoid pure black

### Light-furred cats
- Black void (maximum drama)
- Dark nebula (contrast)
- Grayscale city (iconic)

### Patterned cats
- Simple backgrounds (avoid competition)
- Solid gradients
- Minimal texture

## Common mistakes
- Cat same value as background (lost)
- Background too busy (clutter)
- Background too interesting (competes)
- No contrast (flat)
