# Allowed Variant Zone

`laser_cat_style` has a core identity, but it also has room for variation.

This file defines the boundaries of acceptable variation without losing style identity.

---

# Core principle

The style allows variation in:
- specific motif selection
- specific color combinations
- specific object placement
- specific scale relationships
- specific backgrounds

The style does not allow variation in:
- cat centrality
- collage logic
- tone
- composition modes
- artifact preferences

Variation within structure is creative.
Variation of structure is drift.

---

# Allowed variations

## 1. Motif variation

Within families, specific objects can vary.

### Food family
- pizza, burger, fries, taco, sandwich, bread, donut

### Glam family
- sunglasses, diamonds, hats, jewelry, makeup items

### Sci-fi family

### Co-star family

# Not allowed variations

## 1. Loss of cat centrality
The cat must remain the main subject.

## 2. Loss of collage logic
The image must feel assembled.

## 3. Loss of absurd tone
The image must be stupid, campy, colorful.

## 4. Loss of composition modes
The image must use sticker scatter or single-gag hero.

## 5. Loss of artifact preferences
The image should preserve some roughness or glow.

## 6. Loss of humor engines
The image must activate juxtaposition, scale, power mismatch, or other engines.

---

# Variant zone boundaries

## Core zone
Non-negotiable identity elements.
- cat centrality
- collage logic
- absurd tone
- composition modes
- humor engines

## Flexible zone
Variable elements within structure.
- specific motifs
- specific colors
- specific backgrounds
- specific scales
- specific effects
- specific cats

## Danger zone
Elements that risk drift.
- new motif families
- realistic treatment
- serious tone
- seamless integration
- naturalistic color
- non-cat centrality

---

# Testing variation

When considering a variation, ask:

1. Does it preserve cat centrality?
2. Does it preserve collage logic?
3. Does it preserve absurd tone?
4. Does it use recognizable composition mode?
5. Does it activate humor engines?

If yes to all, the variation is likely allowed.
If no to any, the variation risks drift.

---

# Bottom line

`laser_cat_style` allows:
- variation within structure
- creative combinations
- surprising specific choices

It does not allow:
- variation of structure
- loss of core identity
- drift into adjacent styles

The variant zone is:
**flexible in specifics, rigid in fundamentals**.
