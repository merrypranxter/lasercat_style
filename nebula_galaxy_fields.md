# Nebula Galaxy Fields

Nebula and galaxy backgrounds are a core backdrop family in `laser_cat_style`.

## Characteristics
- Cosmic color clouds
- Star density
- Nebula formations
- Deep space aesthetic
- Color variety (purple, pink, blue, green, orange)

## Use cases
- Space collage cat
- Cosmic food cat
- Sticker scatter mode
- Psychedelic cat
- Giant cat scenes

## Best practices
- High contrast with cat silhouette
- Rich color for object pop
- Sufficient star density
- Not too busy to compete with foreground

## Color families
- Purple/pink nebula
- Blue/green galaxy
- Red/orange stellar
- Rainbow cosmic

## Drift risks
- Too realistic (space photography)
- Too painterly (fine art)
- Too dark (cat lost)
- Too busy (clutter)
