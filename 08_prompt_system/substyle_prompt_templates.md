# Substyle Prompt Templates

Branch-specific prompt templates for each major `laser_cat_style` substyle.

## 1. Cosmic Food Cat

### Template
```
[cat] on [food] in [cosmic background], [food debris], [effect], food-as-throne absurdity, tacky glow, meme collage
```

### Examples
```
Loaf cat on giant pizza slice in purple nebula, floating fries and burger debris, rainbow aura, food-as-throne absurdity, tacky glow
```

```
Upright cat riding taco through star field, cheese and lettuce scatter, neon glow, cosmic food adventure
```

---
