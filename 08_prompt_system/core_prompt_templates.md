# Core Prompt Templates

Reusable prompt templates for common `laser_cat_style` scenes.

## Template 1: Cosmic Food Throne

```
[cat description] seated on giant [food item] like a throne, [expression], [background], surrounded by floating [food debris], [effect], tacky glow, internet meme collage style
```

### Example
```
Blank-faced loaf cat seated on giant cheeseburger throne, smug expression, purple nebula background, surrounded by floating fries and milkshake, rainbow aura, tacky glow, internet meme collage style
```

---
