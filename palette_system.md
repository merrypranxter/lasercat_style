# Palette System

`laser_cat_style` uses specific color palettes that are synthetic, saturated, and internet-native.

## Core palettes

### Rainbow palette
- Red, orange, yellow, green, blue, indigo, violet
- Classic ROYGBIV
- Used in: rainbow puke, rainbow lasers, gradient backgrounds

### Neon palette
- Pink, cyan, magenta, yellow, green
- Blacklight-coded
- Used in: glow effects, drug-cat branch, graphic poster mode

### Cosmic palette
- Purple, pink, blue, green nebula tones
- Stellar colors
- Used in: galaxy backgrounds, space scenes

### Candy palette
- Bright pastels
- High saturation
- Used in: cute-absurd scenes, glam trash

### Synthetic palette
- Unnatural color combinations
- Color-clipped
- Used in: posterized effects, drug-cat branch

## Palette rules
- Prefer synthetic over natural
- Prefer saturated over muted
- Prefer bright over dark
- Prefer contrast over harmony

## Drift risks
- Naturalistic colors
- Muted palettes
- Tasteful restraint
- Color harmony over violence
